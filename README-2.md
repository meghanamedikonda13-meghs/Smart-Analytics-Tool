#  Smart Analytics Tool

## Overview

The Smart Analytics Tool is a generic dataset analysis application that allows users to upload any CSV file and instantly perform exploratory data analysis, missing value analysis, statistical summaries, and dynamic visualizations.

## Features

* CSV File Upload
* Dataset Preview
* Missing Value Analysis
* Statistical Summary
* Dynamic Histogram
* Dynamic Box Plot
* Dynamic Scatter Plot

## Technologies Used

* Python
* Pandas
* Plotly
* Streamlit

## Functionalities

### Dataset Preview

Displays the first few rows of the uploaded dataset.

### Missing Value Analysis

Identifies null or missing values across columns.

### Statistical Summary

Provides descriptive statistics for numerical columns.

### Dynamic Visualizations

* Histogram
* Box Plot
* Scatter Plot

## How to Run

Install dependencies:

python -m pip install -r requirements.txt

Run the application:

python -m streamlit run app.py
